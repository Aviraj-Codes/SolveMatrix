## SolveMatrix 

A modern Sudoku solving web app built with vanilla HTML5, CSS3, JavaScript & Flask. Features real-time validation, intelligent backtracking solver, puzzle generation, and buttery-smooth animations.

## Features

* Lightning-fast puzzle solving with backtracking algorithm
* Live conflict detection (red highlights for row/column/box violations)
* Generate unique puzzles - Easy/Medium/Hard difficulty
* Glassmorphism UI with hover effects & smooth animations
* Keyboard navigation + number pad controls
* Fully responsive - Perfect on desktop & mobile
* Solve ripple effect + celebration animations

## Tech Stack

* Frontend: HTML5, CSS3 (Grid/Flexbox), Vanilla JavaScript
* Backend: Flask + Python Sudoku solver/generator
* API: REST endpoints (/solve, /generate)

## How to Play

* New Puzzle → Generates fresh Sudoku
* Enter numbers → Click cells or use keyboard
* Conflicts → Red highlights show violations
* Solve → Auto-completes with backtracking
* Clear → Reset board

## Controls:

* Arrow keys / WASD → Navigate cells
* Numbers 1-9 → Fill current cell
* Space → Clear cell
* Enter → Solve puzzle
